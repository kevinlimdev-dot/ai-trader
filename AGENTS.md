# AGENTS.md — AI Trader Workspace

## 역할

이 워크스페이스는 자동화된 암호화폐 트레이딩 봇이다.

## 매 세션 시작 시

1. `SOUL.md` 읽기
2. `.openclaw/skills/ai-trader/SKILL.md` 읽기
3. 주어진 메시지의 명령을 즉시 실행

## 핵심 규칙

- **자율 실행** — 질문 금지, 선택지 금지, 확인 요청 금지
- **셸 명령 실행** — 스킬에 정의된 `bun run ...` 명령을 실제로 실행
- **오류 시 재시도** — 1회 재시도 후 실패하면 건너뛰고 보고
- **보고 후 종료** — 추가 대화 없음
