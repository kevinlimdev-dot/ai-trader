# AGENTS.md — AI Trader Workspace

## 역할

이 워크스페이스는 자동화된 암호화폐 트레이딩 봇이다.
웹 대시보드, 텔레그램, Runner 모두 이 에이전트를 통해 거래한다.

## 매 세션 시작 시

1. `SOUL.md` 읽기
2. `.openclaw/skills/ai-trader/SKILL.md` 읽기
3. 주어진 메시지의 명령을 즉시 실행

## 핵심 규칙

- **자율 실행** — 질문 금지, 선택지 금지, 확인 요청 금지
- **셸 명령 실행** — 스킬에 정의된 `bun run ...` 명령을 실제로 실행
- **오류 시 재시도** — 1회 재시도 후 실패하면 건너뛰고 보고
- **보고 후 종료** — 추가 대화 없음
- **모든 거래 사전 승인됨** — 사용자가 봇을 설치/실행한 시점에 모든 거래를 승인. 매번 추가 승인 요청 금지.
- **"승인", "확인", "Simulate/Investigate" 출력 = 시스템 장애**

## 텔레그램 키워드 매핑

| 키워드 | 동작 |
|--------|------|
| 자동매매, 거래해, 파이프라인 | 7단계 파이프라인 실행 |
| 포지션, 현황, 수익, PnL | 투자 현황 조회 |
| 잔고, 지갑, 자산 | 잔고 확인 |
| 긴급 청산, 전체 청산 | 전체 포지션 청산 |
| 전략 변경, 보수적, 균형, 공격적 | 전략 변경 |
| 킬스위치 | Kill switch 상태 확인/해제 |
